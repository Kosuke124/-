このコードはゼミでの活動である地方議会のデータベースを作成する際に作成したコードの一部です。
データベース作成プロジェクトとは、地方議会のデータ（具体的には議員のデータや、会議においてどのような発言をしたかなど）をまとめ、データベースとして研究に利用しやすい形として提供することで政治学の発展に貢献するというようなものです。
データベース全体のテーブルの種類は、多岐にわたり、類似している技術を利用して作成しているため、2つのコードに絞って提出させていただきました。
これらのコードの利用方法としては、スクレイピングによるデータの収集、pandasを利用したデータの結合です。
hatugen.pyについては、地方議会の議事録から発言者と発言内容を取り出し、議員データ（氏名や会派などの議員としての情報）の名前と一致する者とどの会議であるかを結合し、テーブルを作成するコードです。
scraping.pyについては議事録をダウンロードしない場合に直接スクレイピングを行ってデータを作成する場合に利用していたものです。
スクレイピングの方法としてはseleniumとbeautiful soupを主に利用し、コードを作成しました。
